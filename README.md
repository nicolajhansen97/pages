# Freezely - Documentation & Support Pages

This directory contains the privacy policy and support pages for Freezely.

## Files

- `index.html` - Landing page with app overview
- `privacy-policy.html` - Complete privacy policy
- `support.html` - Support page with FAQs and contact info

## Hosting on GitHub Pages

### Option 1: Create a new repository for documentation

1. Create a new repository named `freezely-support` on GitHub
2. Upload these HTML files to the repository
3. Go to Settings → Pages
4. Set Source to "Deploy from a branch" and select "main" branch
5. Your pages will be available at:
   - Landing: `https://nicolajhansen97.github.io/freezely-support/`
   - Privacy: `https://nicolajhansen97.github.io/freezely-support/privacy-policy.html`
   - Support: `https://nicolajhansen97.github.io/freezely-support/support.html`

### Option 2: Use this repository with GitHub Pages

1. Push this `docs` folder to your GitHub repository
2. Go to Settings → Pages
3. Set Source to "Deploy from a branch"
4. Select "main" branch and "/docs" folder
5. Your pages will be available at:
   - Landing: `https://nicolajhansen97.github.io/fridgetracker/`
   - Privacy: `https://nicolajhansen97.github.io/fridgetracker/privacy-policy.html`
   - Support: `https://nicolajhansen97.github.io/fridgetracker/support.html`

## For App Store Submission

Use these URLs in your App Store Connect submission:

- **Privacy Policy URL**: `https://nicolajhansen97.github.io/[repo-name]/privacy-policy.html`
- **Support URL**: `https://nicolajhansen97.github.io/[repo-name]/support.html`

Replace `[repo-name]` with either `freezely-support` or `fridgetracker` depending on which option you choose.

## Updating Content

To update the privacy policy or support page:
1. Edit the HTML files
2. Commit and push to GitHub
3. Changes will automatically deploy (may take a few minutes)

## Contact Information

Remember to update the email address in the HTML files if you want to use a different contact email.
